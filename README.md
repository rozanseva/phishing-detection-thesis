# phishing-detection-thesis
# Разработка методики защиты информации от целевого фишинга в автоматизированной системе организации


## Описание
Разработано программное обеспечение для обнаружения и классификации фишинговых писем. В процессе разработки сформирована структура программного обеспечения для обнаружения и классификации фишинговых писем и обоснован выбор инструментальных средств для его реализации. 
Рассмотрен алгоритм функционирования программного обеспечения и проведено тестирование, подтвердившее возможность его успешной практической реализации.  

## Стек
- **Язык:** Python
- **Библиотеки:** Pandas, IMAP, email, dkim, spf, pickle, pyotp, matplotlib.pyplotи и другие

## Структура проекта
- script.py # Обработка электронных писем
-techii.py # Обучение ML-модели
-model.pkl # Обученная модель
-dataset.csv/ # Размеченный датасет электронных писем
-README.md
